## AI Coding Tools

Tools worth exploring as you level up.

### Free AI Assistants (Browser-Based)

| Tool | URL | Best For |
|------|-----|----------|
| Claude.ai | [claude.ai](https://claude.ai) | Strong at code generation, editing, and long documents |
| ChatGPT | [chat.openai.com](https://chat.openai.com) | Widely available, good at explaining concepts |
| Google AI Studio | [aistudio.google.com](https://aistudio.google.com) | Direct access to Gemini models, free API keys |

### AI Code Editors (Desktop)

These are the next step when you want AI built directly into your coding environment.

| Tool | URL | What Makes It Unique |
|------|-----|---------------------|
| Cursor | [cursor.com](https://cursor.com) | AI-first code editor (VS Code fork) with refactoring, inline editing, and chat |
| Windsurf | [windsurf.com](https://windsurf.com) | AI editor with deep codebase understanding and collaborative "flow" mode |
| GitHub Copilot | [github.com/features/copilot](https://github.com/features/copilot) | AI autocomplete inside VS Code, JetBrains, and other editors |
| Replit | [replit.com](https://replit.com) | Browser-based IDE with AI agent that can build and deploy apps |

### Command-Line Tools

| Tool | URL | What Makes It Unique |
|------|-----|---------------------|
| Claude Code | [github.com/anthropics/claude-code](https://github.com/anthropics/claude-code) | Anthropic's CLI tool -- lives in your terminal, reads your codebase, handles git |

## Deep Dives

Interesting reads from AI companies if you want to understand how these tools work under the hood.

- **[Claude's Constitution](https://www.anthropic.com/news/claudes-constitution)** -- The full set of values and principles Claude was trained on. Fascinating read about how AI companies think about safety and alignment.
- **[Constitutional AI (Research Paper)](https://www.anthropic.com/research/constitutional-ai-harmlessness-from-ai-feedback)** -- The foundational research on training AI with explicit values instead of implicit human feedback.
- **[Claude Code in Action (Free Course)](https://anthropic.skilljar.com/claude-code-in-action)** -- 21-lesson course from Anthropic on using their CLI tool, from basics to advanced.
- **[Addy Osmani's AI Coding Workflow](https://addyosmani.com/blog/ai-coding-workflow/)** -- Practical patterns for working with AI tools from a Google Chrome engineer.
